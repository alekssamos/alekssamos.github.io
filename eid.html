enable inaccessable dialogs

На некоторых веб сайтах есть диалоги. Иногда они бывают полностью недоступны.
Дело в одном единственном атрибуте, это aria-hidden,
который не правильно там используется, его нужно просто удалить и всё.
Для его удаления и создан этот скрипт.

Больше экспериментов с этим скриптом не будет. Теперь он будет выполнять только одну эту функцию и всё.

Для всех сайтов использовать его не целесообразно, теперь он будет работать только на следующих сайтах:

  • nalog.ru
  • rutube.ru
  • iCloud, Apple

Если он ещё где-то нужен - свяжитесь со мной и я добавлю этот сайт.

  1. Сначала установить расширение Tampermonkey
  2. Затем установить в него этот USERSCRIPT
* По моим наблюдениям, через Greasemonkey почему-то скрипт работает хуже,
а Tampermonkey показал себя с лучшей стороны.
